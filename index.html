Shingaku: Deep Dive

EDTECH PLATFORM

Overview: Shingaku.tech was built to revolutionize how college students consume and interact with educational content. The platform integrates Google's Gemini API to provide real-time, context-aware answers and study materials.

Technical Hurdles: We faced immense pressure developing this under strict hackathon deadlines. Integrating the ElevenLabs API for seamless text-to-speech voice features required precise asynchronous handling and robust error boundary setups in React.

Impact: The project received overwhelmingly positive feedback and ultimately secured two major track prizes at Electrothon 8.0 — Best Use of Gemini API and Best Use of ElevenLabs API.
